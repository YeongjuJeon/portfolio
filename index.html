W.Dressroom 반응형 페이지

#HTML5 #CSS3

미디어 쿼리를 사용하여 넓이 1280px 이상일 경우와 1280px 이하, 850px 이하일 경우의 3가지로 제작하였습니다. 또한 슬라이드의 넓이가 페이지의 넓이에 따라 축소 또는 확대 되도록 하였으며 페이지 축소 시 상단 메뉴가 햄버거 메뉴로 변경되도록 제작 하였습니다. 또한 자동슬라이드와 네이게이션, 프로그레스 바를 사용하여 사용자가 쉽게 슬라이드를 이용할수 있도록 디자인 하였습니다.

국립중앙박물관 반응형 페이지

#HTML5 #CSS3 #Jquery

미디어 쿼리를 사용하여 페이지의 넓이가 1440px 이상일 경우, 1440px 이하일 경우, 1200px 이하일 경우와 모바일 화면을 고려하여 700px 이하일 경우로 제작하였습니다. 상단 메뉴의 경우 3Depth로 제작되었으며 마우스를 올렸을경우에는 글자 색의 변화를 주어 사용자가 쉽게 이용할수 있도록 하였습니다. 메인 슬라이드의 pagination으로 현재 페이지가 어디인지 사용자가 확인할수 있으며 하단 슬라이드들의 경우 pagiantion을 fraction으로 설정하였습니다. 상단의 메뉴와 달리 하단에 위치한 메뉴는 사용자가 한눈에 볼 수 있도록 전체 메뉴를 보여주고 있습니다. 1440px 이상일 경우, 1440px 또는 1200px 이하일 경우 태블릿 기기 사용자를 고려한 메뉴와 700px 이하일 경우의 모바일 기기 사용자를 위한 메뉴 총3가지로 제작하였습니다.

에뛰드 반응형 페이지

#HTML5 #CSS3 #Jquery

에뛰드 페이지는 상단 메뉴가 중앙과 오른쪽 햄버거 메뉴 2가지로 구성되어 있습니다. 중앙 메뉴의 경우 마우스를 올릴 경우 하위 메뉴가 나타나며 햄버거 메뉴의 경우 클릭할 경우에 전체 메뉴가 나타나도록 제작하였습니다. 메인 슬라이드는 프로그레스바를 사용하여 사용자의 현재 페이지를 알려주고 있습니다. 에뛰드 페이지는 미디어 쿼리를 사용하여 반응형으로 제작되었습니다. 화면의 크기가 1500~1400px 이하일 경우와 1000px, 800px, 600px, 500px 이하일 경우 제작되어 다양한 환경의 디스플레이에서도 사용자가 불편함 없이 동일한 정보를 얻을수 있도록 제작하였습니다.
